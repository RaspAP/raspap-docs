# Use cases

## What you can do with RaspAP

RaspAP turns your Raspberry Pi into a powerful, flexible network tool. Whether you're traveling, working remotely, or building a home lab, here’s how you can use it:

### Common use cases

- Share a wired connection: Plug into Ethernet and broadcast WiFi to multiple devices.
- Extend weak WiFi: Boost an existing wireless signal to reach more areas.
- Secure public WiFi: Add VPN, AdBlock, and firewall protection when connecting to sketchy networks.
- Create a private LAN: Isolate IoT devices or set up a local-only network.
- Use cellular data: Tether via USB or cellular hotspot to share mobile internet.

### Feature highlights

RaspAP isn’t just a hotspot. It’s packed with tools to make your network smarter:

- Web UI: Configure everything from your browser
- VPN support: [WireGuard](../features-core/wireguard.md) and [OpenVPN](../features-core/openvpn.md) built in
- [AdBlock](../features-core/adblock.md): Block ads network-wide
- [Firewall](../features-core/firewall.md): Control traffic and secure devices
- [Docker](../get-started/docker.md) support: Run RaspAP in containers

### Real-world scenarios

- Hotel Ethernet, including those with [captive portals](../features-insiders/captive.md): Plug in, share WiFi with all your devices, and add a VPN for privacy.
- RV WiFi: Connect to campground WiFi, extend it, and block ads.
- Workshop LAN: Create a local network for smart devices without internet access.
